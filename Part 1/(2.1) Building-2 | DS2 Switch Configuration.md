## RSTP Configuration

```
DS2(config)#spanning-tree mode rapid-pvst
DS2(config)#spanning-tree vlan 100-103 root primary
DS2(config)#interface range ethernet 0/1-2
DS2(config-if-range)#spanning-tree guard root
```
